Subroutine for setting up the GFS hybrid coordinate vertical grid. Files with precalculated coefficients (A_k and B_k) from the "fix" directory in Patrick Tripp's V2 code are used; their filenames start with "global_hyblev". There are only files for 28, 42, 60, 64, and 91 levels. If a different number of levels are specified, an error is returned. Using the A_k and B_k coefficients, the model level pressures, sigma values, and exner function (at interfaces and layer centers) are calculated and returned to the calling procedure.

Parameters
[in]n_levelsnumber of model levels
[in]n_columnsnumber of columns
[in]pres_sfcsurface pressure (Pa)
[out]pres_lpressure at model level centers
[out]pres_ipressure at model interfaces
[out]sigisigma at model interfaces
[out]siglsigma at model layers
[out]exner_lexner function at model level centers
[out]exner_iexner function at model interfaces
[out]a_kGFS hybrid pressure coordinate coefficients
[out]b_kGFS hybrid pressure coordinate coefficients
[out]errorerror code

Algorithm

  • Check to see if the desired number of grid levels is available. If not, return an error code.
  • Open the appropriate file.
  • The first line contains the number of coefficients and number of levels (these should already by known; discard this info)
  • Read in the coefficient data.
  • Calculate interface pressures, sigma, and exner function.
  • Calculate layer center pressures, sigma, and exner function.
