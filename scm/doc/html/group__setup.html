Detailed Description

Contains subroutines to initialize the SCM, including setting the atmospheric state, interpolating initial conditions to the model grid, and patching in a reference sounding above the provided initial conditions.

Subroutine to patch in a reference profile (smoothly) above a given model level.

Parameters
[in,out]last_index_initlast vertical model index that was initialized
[in]n_levels_smoothnumber of levels below the last initialized model level to start smoothing in the reference profile
[in]n_ref_levelsnumber of reference profile levels
[in]n_model_levelsnumber of model levels
[in]ref_presreference profile pressure levels (Pa)
[in]ref_fieldreference profile data
[in]pres_lmodel pressure levels (Pa)
[in,out]model_fieldmodel profile data
[out]errerror code to return to calling procedure (0: no error, 1: reference profile shorter than input, 2: reference profile shorter than model domain)

Algorithm

  • Check for the reference profile being shorter than the input (error code 1), or shorter than the model domain (error code 2).
  • Recalculate the model data profile n_levels_smooth below the last index calculated in order to provided a smoother transition to the reference profile.
    • smooth_frac is the fraction of the reference value to use (remainder is model field already calculated)
    • First, interpolate reference profile to model levels.
    • Then, recalculate field at model level using smooth_frac
  • Above the highest model level specified in the input, just interpolate to the reference profile.
    • If the reference profile does not proceed to the top of the model domain, keep track of the highest level initialized.
    • If the reference profile was shorter than the model domain, extrapolate to fill in the missing values (temporary solution).

Subroutine to interpolate the initial conditions to the model grid and set the state variables.

Parameters
[in]n_input_levelsnumber of levels in the case input data
[in]n_model_levelsnumber of model levels
[in]n_columnsnumber of columns (should be 1 for SCM)
[in]n_smooth_levelsnumber of model levels over which to smoothly transition to the reference profile
[in]n_ref_levelsnumber of levels in the reference profile
[in]ntoztracer array index for ozone
[in]ntcwtracer array index for cloud water
[in]input_prespressure levels for the case input (Pa)
[in]input_qttotal water specific humidity for the case input (kg kg^-1)
[in]input_thetailliquid-ice water potential temperature for the case input (K)
[in]input_qlliquid water specific humidity for the case input (kg kg^-1)
[in]input_qiice water specific humidity for the case input (kg kg^-1)
[in]input_ueast-west wind for the case input (m s^-1)
[in]input_vnorth-south wind for the case input (m s^-1)
[in]input_ozoneozone concentration for the case input (kg kg^-1)
[in]pres_lmodel pressure levels (Pa) (n_columns, n_model_levels)
[in]ref_prespressure levels for the reference profile (Pa)
[in]ref_qvwater vapor specific humidity for the reference profile (kg kg^-1)
[in]ref_ttemperature for the reference profile (K)
[in]ref_ozoneozone concentratin for the reference profile (kg kg^-1)
[out]state_tracerwater vapor specific humidity for the model state, ozone, cloud water [all (kg kg^-1)]
[out]state_ttemperature for the model state (K)
[out]state_ueast-west wind for the model state (m s^-1)
[out]state_vnorth-south wind for the model state (m s^-1)

Algorithm

  • Calculate water vapor from total water, suspended liquid water, and suspended ice.
  • Todo:
    When patching in a reference sounding, need to handle the case when the reference sounding is too short; patch_in_ref checks for the case, but as of now, it just extrapolates where it needs to and returns an error code; error should be handled here or passed up to the main program.
  • For each column, interpolate the water vapor to the model grid.
    • If the input domain does not span the model domain, patch in McClatchey tropical standard atmosphere (smoothly over a number of levels) above.
  • Calculate the input absolute temperature from input pressure, theta_il, ql, and qi.
  • For each column, interpolate the temperature to the model grid.
    • If the input domain does not span the model domain, patch in McClatchey tropical standard atmosphere (smoothly over a number of levels) above.
  • For each column, interpolate the u-wind to the model grid.
    • The standard atmosphere doesn't have wind data; assume zero-gradient above the input data.
  • For each column, interpolate the v-wind to the model grid.
    • The standard atmosphere doesn't have wind data; assume zero-gradient above the input data.
  • For each column, interpolate the ozone to the model grid.
    • If the input domain does not span the model domain, patch in McClatchey tropical standard atmosphere (smoothly over a number of levels) above.
